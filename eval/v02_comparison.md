## Reflection（2026-XX-XX）

### 关键发现
- cosine + Qwen 组合让 5-query 测试集达到 100% Top-1 命中
- 两个变量贡献等量（各 +40%）——说明它们是正交改进
- 但 5 个 query 不足以做出"系统准确率"的结论

### 接下来的工作
- v0.3: 扩大 evaluation 集到 30 query
- 设计原则：覆盖不同问法、不同 KP/EP 类型、包含边界 case
- 预计找到的失败模式：（待 v0.3 测试后填）

### 学到的工程认知
- Embedding 模型选择对中文场景影响很大
- Distance metric 是隐藏的坑（默认 L2 不是你想要的 cosine）
- 5-query 测试是 feasibility check，不是 evaluation
- 真正的 evaluation 需要 ground truth 数据集
